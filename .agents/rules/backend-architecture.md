# 카드 쇼핑몰 백엔드 서버 개발 규칙

## 0. 무조건 한글로 말해야 한다.

## 1. 아키텍처 및 기술 스택
- 이 프로젝트는 프론트엔드(Vue.js)와 분리되어 통신하는 RESTful API 모놀리식 백엔드 서버다.
- 빌드 도구: Gradle
- 프레임워크: Spring Boot (Java)
- 데이터베이스: MySQL

## 2. 데이터베이스 접근 원칙 (필독)
- 단순 CRUD 로직 및 기본 엔티티 관리는 **Spring Data JPA**를 사용한다.
- 복잡한 검색 필터, 상품 다중 테이블 조인, 통계 쿼리 등은 반드시 **MyBatis**를 사용한다. 

## 3. 코드 스타일 및 협업
- 프론트엔드 담당자와 원활하게 협업하기 위해, 모든 API 응답(Response) 형식은 공통 DTO 구조를 만들어 일관되게 반환한다.
